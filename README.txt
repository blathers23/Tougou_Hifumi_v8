> pytorch为GPU版本，并且只能使用NVIDIA的GPU，需要较新的显卡驱动以及CUDA的支持。

> 模型学习流程 Train -> Fit -> Main

> 开发使用的硬件为:
  -----------------------------------------
  CPU   AMD   Ryzen7 3700X
  GPU	NVIDIA GeForce GTX1050Ti
  -----------------------------------------
  较差的配置可能导致运行速度的下降

> 推荐的程序改进方向：
  ----使用C++改写程序以获得更快的运行速度，增加模拟的次数从而提升模拟的稳定性
  	可能存在的缺陷：神经网络模型并不够准确，导致提升模拟次数也不能使程序棋力有明显的提升

  ----使用训练好的模型文件，进行深度强化学习
  	可能存在的缺陷：深度强化学习10次进行8次失败
